A sleek, responsive task management application built with React. This project focuses on a smooth user experience, featuring an infinite carousel for task navigation and persistent data storage.

*Features*
Endless Task Carousel: A custom-built, infinite scrolling component that displays 3 tasks per row, optimized for desktop and tablet views.

Persistent Storage: Full integration with localStorage to ensure your tasks and UI preferences (like Dark Mode) remain even after a page refresh.

Due Date Tracking: Each task includes a dedicated due date field to help you stay on top of deadlines.

Dark Mode Support: A toggleable theme that saves your preference locally.


*Tech Stack*
Frontend: React (Hooks, Functional Components)

Styling: Vanilla CSS (Custom Properties, Flexbox, Grid)

Storage: Browser LocalStorage API
